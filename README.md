FLAMES is a lightweight web-based game that calculates relationship results using the traditional FLAMES algorithm. Users simply enter two names, and the game instantly displays the result in a clean and interactive interface.
The project is developed using HTML, CSS, and JavaScript, hosted for free using GitHub Pages, and is fully responsive.
